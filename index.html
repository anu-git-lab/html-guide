What is the Internet?

The Internet is a global network that connects computers and devices around the world.

How the Internet Works

Client-Server Model

Clients request resources and servers provide them.

HTTP

HTTP is the protocol used for communication between web browsers and web servers.

DNS

DNS converts domain names into IP addresses.

Main Internet Services

  • Email
  • Web Browsing
  • Cloud Storage
  • Video Streaming
  • Online Gaming

Internet in Daily Life

The Internet supports communication, education, banking, shopping, entertainment and business.

Role of Web Browsers

Web browsers such as Chrome, Edge and Firefox retrieve, display and navigate websites.

What is DNS and Why It Is Important

DNS makes websites easy to access because users can remember names instead of IP addresses.
